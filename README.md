## About

<table>
<tr>
<td>

這個 Hawaii Travel 是一個模擬夏威夷旅遊租借服務的網站，沒有實作任何功能性，主要用於練習 HTML 5, CSS 3 以及 SCSS。該網站的特色有:

- 實作 FlexBox, Media Query 自適應 (RWD) 網頁
- 實作黏性導覽列 (sticky navigation bar)
- 實作一頁式展示頁面 (cover)
- 實作圖片縮放特效 (image scale effect)
- 實作頁尾 (footer)

Built With:

* HTML 5
* CSS 3
* SCSS

</td>
</tr>
</table>
